# Courier Management System

## Project Structure

This is a MERN stack application with separate backend and frontend folders.

### Installation

**Backend:**
```bash
cd backend
npm install
```

**Frontend:**
```bash
cd frontend
npm install
```

### Running the Application

**Backend:**
```bash
cd backend
npm start
```

**Frontend:**
```bash
cd frontend
npm run dev
```

## Note
- Backend dependencies include all server-side packages (Express, MongoDB, PDF generation, Excel generation, etc.)
- Frontend dependencies include React and related UI libraries
- No root-level node_modules folder - each application manages its own dependencies
